方言的特殊类型

创建新方言

GORM 官方支持以下几种方言：sqlite, mysql, postgres, mssql.

你可以通过创建一个新的方言来为其它数据库提供支持。 当你创建一个新方言的时候，你必须实现 the dialect interface 接口。

某些数据库可能兼容 mysql 或 postgres 方言，此时你可以直接使用现有方言。

方言的特殊类型
